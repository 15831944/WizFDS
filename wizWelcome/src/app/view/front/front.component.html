ABOUT
WizFDS is open-source project released under GNU v3.0 license founded by F&K Consulting Engineers Ltd.
Our software is Graphical User Interface (GUI) for Fire Dynamics Simulator (FDS).
We are using it everyday to enrich our work.
We believe the workflow of the FDS process can be refined through integration and automation.
We aim at creating the platform based on CAD integration, semantic Editor, domain knowledge and other.
Our goal was not to implement all FDS functions. We focused on the most common.
https://github.com/fkce/WizFDS
HOW IT WORKS
We created flexible web-application, so you will never have to search for your input files.
Our goal was not to build new Graphical Engine - it's better to use professional CAD softwares i.e. AutoCAD, BricsCAD.
So we created plugin to help creating FDS geometrical representation.
